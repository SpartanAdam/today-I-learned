# Create tables

Tables can be easily formatted in Markdown using standard characters, for example:

```python
|    A |   B |   C |    D |   mean |
|-----:|----:|----:|-----:|-------:|
| 74   |  33 |  96 | 66   |  67.25 |
| 69   |  17 |  95 | 38   |  54.75 |
| 45   |   8 |  22 |  3   |  19.5  |
| 96   |  95 |  86 | 22   |  74.75 |
| 63   |  82 |  21 | 40   |  51.5  |
| 69.4 |  47 |  64 | 33.8 |  53.55 |
```
